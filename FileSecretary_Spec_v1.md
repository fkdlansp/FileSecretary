## 2. 전체 기능 스코프

### 2-1. 파일 정리 탭

| 기능 | 상세 |
|---|---|
| 대상 폴더 등록 | 드래그앤드롭 또는 버튼 클릭(Finder 열기)으로 여러 폴더 등록 |
| 대상 폴더 제거 | 목록의 빨간 − 버튼으로 제거, 마우스 hover 시 "제외" 툴팁 |
| 카테고리 추가 | 별도 모달 창에서 이름 + 분류 조건 설정 후 추가 |
| 카테고리 수정 | 카드의 파란 ✎ 버튼 → 기존 값이 채워진 수정 모달 |
| 카테고리 제거 | 카드의 빨간 − 버튼, hover 시 "제외" 툴팁 |
| 카테고리 순서 변경 | 드래그로 순서 변경, 변경 시 번호 전체 재정렬 |
| 분류 조건 — 키워드 | 파일명 내 특정 키워드 포함 여부 (쉼표로 구분 입력) |
| 분류 조건 — 파일 타입 | 이미지 / 동영상 / 문서 / 어도비 중 복수 선택 |
| 분류 조건 — AND/OR | 키워드+타입 동시 사용 시 AND(둘 다 해당) 또는 OR(하나라도 해당) 선택 |
| 타입 확장자 팝업 | 카테고리 모달 내 타입 pill에 마우스 hover 시 해당 확장자 목록 팝업 |
| 기타 카테고리 | 항상 마지막 고정, 드래그/삭제 불가, 분류 안 된 파일 처리 전 확인 다이얼로그 |
| 카테고리 중복 처리 | 여러 카테고리에 해당하는 파일 발견 시 개별 확인 다이얼로그 |
| 폴더 생성 형식 | `01_카테고리명` 형식 (번호 2자리 + 언더스코어 + 이름) |
| 기존 폴더 재사용 | 같은 이름 폴더 존재 시 새로 만들지 않고 해당 폴더 사용 |
| 출력 폴더 등록 | 최대 4개, 드래그앤드롭 또는 Finder 버튼으로 등록 |
| 출력 폴더 — 메인 | 첫 번째 폴더가 메인, 미지정 카테고리는 메인 폴더의 기타로 이동 |
| 카테고리별 출력 지정 | 각 카테고리 카드 하단 드롭다운으로 출력 폴더(A/B/C/D) 또는 개별 모드 선택 |
| 개별 모드 | 출력 폴더 미등록 또는 카테고리별 "개별 모드" 선택 시 각 대상 폴더 안에서 분류 |
| 수동 정리 실행 | "지금 정리하기" 버튼 |
| 중복 파일 처리 | 정리 실행 시 다이얼로그: 번호 추가(기본값) / 덮어쓰기 / 건너뛰기 |
| 되돌리기 | 히스토리 스택, `↩ 되돌리기 (n)` 버튼, n=0이면 비활성화 |
| 원 클릭 다운로드 정리 | ~/Downloads 고정, 개별 모드, 파일 타입 기준, 넘버링 없음 |
| 제외 목록 | 키워드 또는 확장자명 기준, OR 처리, 환경설정 메뉴에서 편집 |
| 정리 이력 로그 | 정리 실행마다 텍스트 파일로 자동 저장 (~/Library/Logs/FileSecretary/) |

### 2-2. 파일명 편집 탭

| 기능 | 상세 |
|---|---|
| 폴더 등록 | 드래그앤드롭 또는 버튼으로 폴더 추가, 하위 파일 목록 표시 |
| 파일 목록 | 고정 높이 + 스크롤 (일정 개수 초과 시 스크롤) |
| 순서 변경 | 행 전체 드래그, 끼워넣기 방식 (파인더 스타일), 위/아래 절반 감지해 삽입 위치 표시 |
| 넘버링 자릿수 | 1~5자리 세그먼트 버튼으로 선택 |
| 넘버링 시작 번호 | 시작 번호 직접 입력 (기본값 1, 양의 정수만 허용), 자릿수에 맞춰 0 패딩 적용 |
| 파일명 입력 | "파일명" 레이블 + 인라인 입력창 (비우면 기존 파일명 유지) |
| 파일명 통일 체크박스 | ON: 모든 파일에 공통 파일명 적용 (`넘버링_파일명.확장자`) |
| 파일명 통일 OFF | 공통 파일명 있으면 `넘버링_파일명(원본파일명).확장자`, 없으면 `넘버링_원본파일명.확장자` |
| 인라인 편집 | 파일명 열 더블클릭 → 인라인 텍스트 편집 (Enter 확정 / Escape 취소) |
| 파일명 통일 ON 시 | 인라인 개별 편집 비활성화 |
| 미리보기 | 각 행 우측에 최종 파일명 초록색으로 실시간 표시 |
| 미리보기 헤더 | 넘버링 / 파일명 / 미리보기 3단 열 헤더 |
| 초기화 | 모든 커스텀 파일명 초기화, 파일명 통일 체크 해제 |
| 적용 | 리네임 실행 |

### 2-3. 공통

| 기능 | 상세 |
|---|---|
| 디스크 용량 표시 | 탭 위 고정 영역: 볼륨명 / 전체 / 사용 / 잔여 + 프로그레스 바 |
| 디스크 용량 색상 | 75% 이상 주황, 90% 이상 빨강 |

| 환경설정 | 상단 메뉴바 > 환경설정: 제외 목록 편집 / 로그 파일 열기 / 기본값 초기화 |

## 3. UI 설계

### 3-1. 화면 상태 구조

앱은 하나의 창, 두 가지 상태로 동작한다.

```
상태 A (초기)                    상태 B (확장)
작은 창 (약 220×180pt)    →      대각선 확장 (오른쪽+아래)
탭 + 드래그앤드롭 영역만           좌/우 패널 분리 레이아웃
```

### 3-2. 화면 A — 초기 상태

```
┌──────────────────────┐
│ ● ● ●  File Secretary│
├──────────────────────┤
│ 파일 정리 │ 파일명 편집│  ← 탭 (화면 A에서도 전환 가능)
├──────────────────────┤
│                      │
│   ┌ ─ ─ ─ ─ ─ ─ ┐   │
│   │      ↓       │   │  파일 정리 탭: "폴더를 여기에 드롭하세요"
│   │  드롭 안내    │   │  파일명 편집 탭: "파일 또는 폴더를 여기에 드롭하세요"
│   └ ─ ─ ─ ─ ─ ─ ┘   │
│                      │
└──────────────────────┘
```

### 3-3. 화면 B — 확장 상태 (파일 정리 탭)

```
┌──────────────────────────────────────────────────────────────┐
│ ● ● ●                     File Secretary                      │
├──────────────────────────────────────────────────────────────┤
│  Macintosh HD   전체 512GB   사용 347GB   잔여 165GB   68% ████░│  ← 디스크 바
├──────────────────────────────────────────────────────────────┤
│ 파일 정리 │ 파일명 편집                                          │
├────────────────────┬─────────────────────────────────────────┤
│ 대상 폴더           │ 카테고리                                   │
│ ─────────────────  │ ──────────────────────────────────────── │
│ 01 ~/Downloads  ●  │ ⠿ 01 어도비 작업  [타입]  ✎ ●             │
│ 02 ~/Desktop    ●  │    타입: 어도비          출력 [A: Creative ▾]│
│                    │                                           │
│ + 폴더 추가 / 드롭  │ ⠿ 02 디자인 시안  [키워드+타입]  ✎ ●        │
│ ─────────────────  │    키워드: 시안  AND  타입: 이미지           │
│ 출력 폴더 (최대 4개) │    출력 [B: 시안 ▾]                        │
│ ● A ~/Creative 메인 │                                           │
│ ● B ~/Dropbox/시안  │ ⠿ 03 영상 클립  [타입]  ✎ ●               │
│ + 출력 폴더 추가    │    타입: 동영상          출력 [A: Creative ▾]│
│ ─────────────────  │                                           │
│ 미지정된 파일은      │ — 기타  [항상 고정]                         │
│ 메인 폴더의 기타    │    분류되지 않은 파일 → 확인 후 이동           │
│ 폴더로 이동됩니다.  │                                           │
│ ─────────────────  │ + 카테고리 추가                             │
│ 원 클릭 다운로드    │ ─────────────────────────────────────────  │
│ 폴더 정리           │              ↩ 되돌리기 (2)  [지금 정리하기] │
└────────────────────┴─────────────────────────────────────────┘
```

#### 왼쪽 패널 상세

| 요소 | 동작 |
|---|---|
| 대상 폴더 목록 | 번호 + 경로 + 빨간 − 버튼 (hover: "제외" 툴팁) |
| + 폴더 추가 / 드롭 | 클릭 시 Finder 열기 / 드래그앤드롭 동일 동작 |
| 출력 폴더 목록 | 색상 점(A=파랑 B=초록 C=주황 D=보라) + 라벨 + 경로 + 메인 뱃지(첫 번째) |
| + 출력 폴더 추가 | 4개 초과 시 비활성화 |
| 원 클릭 다운로드 정리 | ~/Downloads 고정, 파일 타입 기준 명시, 클릭 시 즉시 실행 |

#### 오른쪽 패널 상세

| 요소 | 동작 |
|---|---|
| 카테고리 카드 | 드래그 핸들(⠿) + 번호 + 이름 + 조건 뱃지 + ✎ + − 버튼 |
| 카드 하단 | 조건 태그 + 출력 폴더 드롭다운 한 줄 배치 |
| 출력 드롭다운 | 개별 모드 / A(메인) / B / C / D 선택 |
| 되돌리기 버튼 | #ff9800 주황, n=0 시 비활성화 |
| 지금 정리하기 | #4caf50 초록 |

#### 파일명 통일 체크박스 동작

| 상태 | 미리보기 형식 |
|---|---|
| OFF + 파일명 입력 없음 | `0001_원본파일명.확장자` |
| OFF + 파일명 입력 있음 | `0001_파일명(원본파일명).확장자` |
| ON | `0001_파일명.확장자` (원본 파일명 무시) |

### 3-5. 다이얼로그

#### 중복 파일 처리 (정리 실행 시 항상 표시)
- 번호 추가 (기본값): `사진.jpg → 사진 2.jpg`
- 덮어쓰기
- 건너뛰기

#### 기타 폴더 확인
- 분류되지 않은 파일 발견 시 표시
- "기타로 이동" 또는 "건너뛰기"

#### 카테고리 중복 파일 확인
- 여러 카테고리에 해당하는 파일별로 개별 표시
- 이동할 카테고리 선택 또는 "이 파일만 건너뛰기"

### 3-6. 카테고리 추가/수정 모달

```
┌──────────────────────────────┐
│  카테고리 추가                 │
│                              │
│  카테고리명  [____________]   │
│                              │
│  분류 조건                    │
│  □ 키워드  [______________]  │
│  □ 파일 타입                  │
│     [이미지][동영상][문서][어도비] │
│     (hover 시 확장자 팝업)      │
│     조건 처리: [AND ▾] 둘 다 해당│
│                              │
│              [취소]  [추가]   │
└──────────────────────────────┘
```

- 파일 타입 체크 해제 시 타입 pill 비활성화
- 키워드+타입 동시 체크 시 AND/OR 선택 영역 표시
- pill hover 시 해당 확장자 목록 팝업 표시

### 3-7. 상단 메뉴바

```
File Secretary   파일   프리셋   환경설정   도움말
```

| 메뉴 | 항목 |
|---|---|
| 프리셋 | 프리셋 저장 / 프리셋 불러오기 |
| 환경설정 | 제외 목록 편집... / 로그 파일 열기 / 기본값으로 초기화 |

## 4. 분류 규칙 설계

### 4-1. 파일 타입별 확장자

| 타입 | 확장자 |
|---|---|
| 이미지 | jpg, jpeg, png, gif, bmp, tiff, heic, webp, svg |
| 동영상 | mp4, mov, avi, mkv, wmv, m4v |
| 문서 | pdf, doc, docx, ppt, pptx, xls, xlsx, csv, txt, md, hwp |
| 어도비 | psd, psb, ai, eps, prproj, aep, aet, indd, indt, idml, xd, lrcat |

> 어도비 상세: 포토샵(psd, psb), 일러스트레이터(ai, eps), 프리미어 프로(prproj), 애프터 이펙트(aep, aet), 인디자인(indd, indt, idml), XD(xd), 라이트룸(lrcat)

### 4-2. 원 클릭 다운로드 폴더 정리 기본 규칙

카테고리 설정이 없을 경우 파일 타입 기준으로 자동 분류, 넘버링 없음.

| 기본 폴더명 | 해당 타입 |
|---|---|
| 이미지 | 이미지 타입 전체 |
| 동영상 | 동영상 타입 전체 |
| 문서 | 문서 타입 전체 |
| 어도비 | 어도비 타입 전체 |
| 기타 | 위 타입 외 전체 |

### 4-3. 분류 우선순위

1. 카테고리 번호 순서대로 평가
2. 첫 번째로 매칭되는 카테고리로 분류
3. 여러 카테고리에 동시 매칭 시 중복 처리 다이얼로그 표시
4. 어디에도 매칭 안 되면 기타 카테고리로 이동 (확인 후)

### 4-4. 제외 목록

- 키워드 기준: 파일명에 해당 문자열 포함 시 제외
- 확장자 기준: 해당 확장자 파일 전체 제외
- OR 처리: 둘 중 하나라도 해당하면 제외
- 기본 제외 항목: `.DS_Store`, `.gitignore`

### Phase 5 — 파일명 편집 탭

**목표:** 파일명 편집 탭 전체 구현

**작업 범위:**
- `FileRenameView.swift` — 탭 전체 레이아웃
- `RenameRowView.swift` — 3단 그리드 행 (넘버링 / 파일명 / 미리보기)
- 넘버링 자릿수 세그먼트 (1~5자리)
- 넘버링 시작 번호 입력 (기본값 1, 양의 정수 검증, 자릿수 초과 시 자리수 자동 확장)
- 파일명 입력 + 파일명 통일 체크박스
- 파일명 통일 ON/OFF 동작 (미리보기 실시간 반영)
- 더블클릭 인라인 편집 (Enter 확정 / Escape 취소)
- 파일명 통일 ON 시 개별 편집 비활성화
- 목록 고정 높이 + 스크롤
- 드래그 끼워넣기 (위/아래 절반 감지, 파란 선 표시)
- `FileRenamer.swift` — 실제 파일명 변경 실행
- 초기화 / 적용 버튼

**완료 기준:**

- [ ] 드래그 끼워넣기 순서 변경
- [ ] 파일명 통일 체크박스 동작
- [ ] 인라인 편집 및 미리보기 실시간 반영
- [ ] 시작 번호 입력 시 미리보기 번호 즉시 반영
- [ ] 적용 시 실제 파일명 변경
